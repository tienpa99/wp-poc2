Premium Blocks for Gutenberg helps you build amazing pages with the new WordPress Gutenberg editor.

== Description ==
Premium Blocks for Gutenberg helps you create amazing looking websites using the new WordPress Editor Gutenberg. All blocks have been built with performance in mind. Also all Gutenberg blocks included are RTL ready. Also Premium Blocks for Gutenberg is translation ready so you can easily translate it to your language.


Premium Blocks for Gutenberg plugin is continually evolving with regular weekly updates that add exciting new features and enhancements. The plugin’s blocks offer a wealth of customization options, giving you complete control over every aspect of your website’s design.


### Useful Links
[Support](https://my.leap13.com/contact-support/) | [Docs](https://premiumblocks.io/docs/?utm_source=wp-repo&utm_medium=link&utm_campaign=readme)



### Equal Height Feature
Now, you can make your columns, Blocks or any selector you want the same height with just ONE click using Equal Height feature in parent container block -> Advanced tab -> Equal Height > Enable Equal Height option.



### Entrance Animation for Gutenberg Blocks Feature
Make your content stand out with the new Entrance Animation feature in Premium Blocks for Gutenberg. With this feature, you can add captivating entrance animation to your blocks, columns, and sections.



### Premium Global Settings
Global Settings is a new tool to control colors and typography of your website elements to be more consistently, which apply global styles for all premium blocks and the native Gutenberg blocks, and includes convenient breakpoint settings for even more control.



### Asset File Generation Feature
generates a separate CSS file for your blocks and save it in the uploads folder. This can significantly improve the loading speed of your website, resulting in a better user experience.

== Installation ==

= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= Installation Instructions =

- First make sure that Gutenberg is installed, As this plugin works only with it.
- Download the plugin then Upload it to the plugin folder: /wp-content/plugins/ or install it through the WordPress plugins screen directly.
- Activate the plugin through the ‘Plugins’ screen in WordPress

= Now, you can be a contributor =
If you want to contribute, go to our [Premium Blocks for Gutenberg GitHub Repository](https://github.com/Leap13/premium-blocks-for-gutenberg) and see where you can help.

== Frequently Asked Questions ==
= Is this a standalone Plugin? =

No. You cannot use Premium Blocks without Gutenberg as it's a blocks plugin for Gutenberg.

= Does it work with any WordPress theme? =

Yes it will work with any WordPress theme as long as you are using Gutenberg as a page editor.
